Vibernate 2.0

  • Platform: Android
  • Duration: March 2015 - May 2015
  • Tools: IntelliJ, Git, Android SDK

The Vibernate app lets you schedule your phone to automatically switch from ringtone to vibrate/silent and back again!

Got lectures every Monday and Friday from 8-10? Simply set up a Vibernate timer and forget about interrupting class for the rest of the year! And don't worry about missing any calls. Vibernate will conveniently set your phone back to its normal ringtone right after class. This app will keep your phone quiet only when you want it to be!


Responsibilities

  • Implemented the core functionality of the app that controls the device’s ring tone mode by issuing Pending Intents and using Broadcast Receivers to trigger actions at a specified time.
  • Implemented a tutorial for new users that gets triggered when the app is launched for the first time. The tutorial goes over the key features of the app.
  • Implemented the activity which allows users to create and modify timers.
  • Worked with a designer in the design of the UX/UI of the app
  • Published the app to the Play store.

Bruce Guitars

  • Platform: Web
  • Duration: Summer 2014
  • Tools: HTML, CSS, Javascript, JQuery, Git

Bruce Guitars is a local Vancouver premium boutique guitar shop created mainly to help fellow artists and musicians find their sound. This project is a redesign of the existing Bruce Guitars website as part of MyPlanet's fellowship program.

After talking to Bruce and conducting interviews with some of his customers, we noticed a clear disconnect in Bruce's site experience and the intimate experience his customers find in-store. With this in mind, we rethought the flow of the site’s content to be more intuitive and presents Bruce's passion for his business and his customers.


Responsibilities

Where's The Food

  • Platform: Web
  • Duration: Fall 2014
  • Tools: HTML, CSS, Javascript, JQuery, Git, Sublime Text

WheresTheFood is a web application that allows users to view local food vendors in Vancouver. We are using Food Vendors Data Catalogue provided by the city of Vancouver together with Google Maps API to create an easy-to-navigate map view where each marker on the map represents a food vendor location. Users can click on each marker to view more details about the food vendor as well as their hours of operation which are fetched from Yelp API.

Each user can log into their Google Account to add and edit personal favourite food truck locations. They can also attach a short memo for each food truck for things like their orders and special deals. This information is stored and loaded the next time the user logs back in. Finally we are integrating Facebook to allow users to share their favourite food trucks with friends on Facebook.

Given the size of the project, we decided on using BackboneJS because of it's simplicity as well as using Firebase for real-time data sync and storage.


Responsibilities

  • Parsed JSON dataset of local food trucks in Vancouver and stored on the server-side along with users' information.
  • Implemented Facebook integration and login to allow users to like, favourite, add comments and memos for different food trucks.
  • Implemented a search functionality that allows users to look up food trucks and filter based on ratings and distance from the user
  • Integrated Google Maps API to plot food trucks on an interactive map
